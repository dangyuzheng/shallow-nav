# DESIGN.md

## 品牌与气质
- 品牌名：浅途 Nav
- Slogan：轻浅行路，直达实用良站
- 气质意象：清晨薄雾中的小径，通向一个个精心挑选的目的地。轻盈、通透、不冗余。
- 关键词：轻浅、直达、实用、精致

## Design Tokens

### 色彩
- 主色：柔和靛蓝 #6366f1（Indigo 500）— 如清晨天际的微光
- 辅色：淡紫 #a78bfa（Violet 400）— 远山薄雾
- 背景：极浅灰白 #f8fafc — 如晨雾散后的天空
- 卡片底色：纯白 #ffffff，带轻微毛玻璃效果
- 文字主色：#1e293b（Slate 800）
- 文字辅色：#64748b（Slate 500）
- 分类标签色：每个分类独立配色，柔和渐变

### 字体
- 字体族：Noto Sans SC（Google Fonts CN 域）
- 标题：600 weight, 24-32px
- 正文：400 weight, 14-16px
- 辅助文字：300 weight, 12-13px

### 圆角与阴影
- 卡片圆角：16px
- 按钮圆角：10px
- 卡片阴影：0 1px 3px rgba(0,0,0,0.06), 0 4px 12px rgba(0,0,0,0.04)
- 悬浮阴影：0 4px 16px rgba(99,102,241,0.12), 0 8px 24px rgba(0,0,0,0.06)

### 动效
- 卡片悬浮：上移 4px + 阴影增强，transition 300ms ease
- 页面切换：fade + slide-up
- 分类切换：平滑滚动 + 高亮过渡
- 收藏动画：心跳缩放效果

## 布局
- 最大宽度：1280px 居中
- 卡片网格：PC 端 4 列，平板 3 列，手机 1-2 列
- 顶部固定导航栏（毛玻璃效果）
- 分类标签横向滚动（移动端）

## 设计禁忌
- 不要深色/暗黑主题
- 不要过于鲜艳的荧光色
- 不要复杂的 3D 效果
- 不要过多的装饰元素干扰信息获取
